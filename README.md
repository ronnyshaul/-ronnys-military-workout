# Ronny's Military Workout

This is a mobile-first installable web app.

## Run locally
Open index.html in a browser, or use a simple local server:

python3 -m http.server 8000

Then visit http://localhost:8000

## Install on iPhone
After deploying to a free host such as GitHub Pages, Netlify, or Cloudflare Pages:
1. Open the app in Safari.
2. Tap Share.
3. Tap Add to Home Screen.

## Included
- Full Week 1 with six workouts
- 10 to 15 minute workout format
- Animated exercise demonstrations
- Voice countdowns and exercise transitions
- Local progress tracking
- 12-week program structure
- Military ranks, streaks, settings, and notes

The exercise demonstrations are animated placeholders, not filmed videos. Real MP4 clips can replace them later.
